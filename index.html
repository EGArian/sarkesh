Welcome to Sarkesh home page.

sarkesh is php framework that support server side events.for example click on button and run a method of class in server can be doing in this framework . With sarkesh you do not need know about html and css. all controls on sarkesh will be draw with php. in sarkesh you can use raintpl template engine for style and other things but without that you have many controls that can use for design like ctr_button,ctr_textarea,ctr_form and etc.
